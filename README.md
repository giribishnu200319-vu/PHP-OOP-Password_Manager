  2. This project presents the development of a Password Manager System using PHP Object-Oriented Programming (OOP) concepts and MySQL database management.
     The main purpose of the system is to provide secure password generation, encryption, storage, and management for users.
     The application allows users to register and log into the system securely using hashed passwords. After authentication, users can generate strong passwords based on customized parameters such as uppercase letters, lowercase letters, numbers, and special characters.
     The generated or manually entered passwords are encrypted using the AES-256-CBC encryption algorithm before being stored in the MySQL database. The project also implements secure database communication using prepared statements to reduce SQL injection risks.
     Object-oriented programming principles such as classes, methods, encapsulation, and modular programming are applied throughout the system to improve code organization and maintainability.
     The system includes essential features such as user registration, login authentication, password generation, password encryption/decryption, and password storage management.
     This project demonstrates practical implementation of PHP OOP, database integration, cybersecurity principles, and web application development techniques in a real-world scenario.
